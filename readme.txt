github的pull request练习

1 每位同学fork本代码库（即wjl8026/github-test-project-1）。

2 在自己fork的代码库（即your-github-id/github-test-project-1)中，新创建一个yourname.txt或your-github-id.txt的文本文件（即用你的姓名或github帐户名作为文件名的txt文件）。txt文件内容随意，但是请注意文明用语。

3 发起一个pull request，请求同步修改本代码库（即wjl8026/github-test-project-1）。
